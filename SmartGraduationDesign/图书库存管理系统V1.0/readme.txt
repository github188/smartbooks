

开发环境：
1.vs210
2.Crystal Report 13.0
3.sqlserver 2005

说明：vs10版本默认不带水晶报表组件,需要自行到Sap网站下载网站。
水晶报表for vs10下载地址(278.01MB)：http://downloads.businessobjects.com/akdlm/cr4vs2010/CRforVS_13_0.exe


根据情况您可能还需要以下组件：
1.安装net4.0框架
  下载地址:http://download.pchome.net/development/developtools/translater/detail-171222.html
2.安装net3.5 sp1补丁包(在线安装，时间根据网速情况可能比较长)
  下载地址:http://www.microsoft.com/downloads/zh-cn/details.aspx?familyid=ab99342f-5d1a-413d-8319-81da479ab0d7


运行说明:
1.附加数据库到sql2005.
2.修改配置文件(GoodsReportManage.exe.config)数据库连接字符串,将相应数据库服务器名称、数据库、用户名、密码替换。
3.运行程序。
  用户名:admin
  密码:123456
  身份:管理员
